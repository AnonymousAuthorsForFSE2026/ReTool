纽约证券交易所拍卖市场买入和卖出规则（2015）

Rule 63 对于以“发行时”为基础买卖的证券，买卖盘应仅在“发行时”进行，即交易所确定的发行时交割。对于以“分发时”为基础买卖的证券，买卖盘应仅在“分发时”进行，即交易所确定的分发时交割。当某只股票的“发行时”或“分发时”交易暂停，而该股票继续以“常规方式”交易时，所有以“发行时”或“分发时”为基础买卖该股票的订单，应于暂停该等“发行时”或“分发时”交易的前一天收市时失效，除非交易所另有规定。

Rule 64 (a) (i) 除下文 (ii) 项规定外，以“已发行”方式交易的证券的买卖，应以“常规方式”进行，即在合约签订日后的第三个工作日交割。此外，交易所可允许交易所不时决定的任何其他结算指示和期限。（有关其他允许的结算指示，请参阅规则 14（订单的非常规方式结算指示）。）
(ii) 在认购截止日期前的第二和第三个工作日，认购权的买卖，应仅以“次日”进行，即在合约签订日后的下一个工作日交割；且在认购截止日期前一天，仅以“现金”方式进行。
(b) 所有以非“常规方式”结算的交易，如果其价格与“常规方式”买入价或卖出价相差超过 0.10 个点，则必须经场内官员批准，但年度最后一周除外，届时仅当成交价格与“常规方式”买入价或卖出价相差超过 0.25 个点时才需要场内官员批准。在考虑是否批准时，场内官员应考虑交易价格相对于“常规方式”市场是否合理。
(c) 所有“卖方期权”交易，如果交割期为 2 至 60 个工作日，则应仅以日历日为单位报告到报价带上。例如，如果交易在 6 个工作日内结算，则打印为“卖方 8”，除非中间有节假日（在这种情况下，它将打印为“卖方 9”）。计算“卖方期权”交易的打印价格时，周末和节假日也计算在内，交易日期不包含在内。以日历日打印的“卖方期权”交易的结算日期不能与正常的三个工作日的“常规方式”结算一致。

规则 66 对于以“已发行”方式交易的美国政府证券，买卖双方只能以该证券的“常规方式”进行，即在交易日的次一工作日交割。此外，交易所可随时决定延长交割期限。

Rule 67_1 如果试点证券的价格在任何特定工作日的常规交易中跌破 1.00 美元，该试点证券将继续按照相关规则进行交易，如同试点证券的价格未跌破 1.00 美元一样。但是，如果试点证券在任何特定工作日的收盘价低于 1.00 美元，该试点证券将从其各自的试点测试组移至控制组，并可在试点期剩余时间内以交易所规则目前允许的任何价格增量进行报价和交易。

Rule 67_2 担任做市商的成员机构应收集并向其指定审查机构（“DEA”）传输数据，这些数据涉及为巩固其做市商地位而在任何试点前数据收集证券和试点证券交易中心开展的活动，包括在全国性证券交易所以外执行已结算或已达到结算日的交易的交易中心。做市商应于T+4日美国东部时间中午12:00之前，按照其DEA要求的格式传输此类数据。

Rule 67_3 任何会员组织不得以0.05美元以外的增量显示、排序或接受任何人提供的任何可显示或不可显示的买入价、卖出价、订单或意向指示。但是，以全国最佳买卖价 (NBBO) 或专业最佳买卖价 (PBBO) 中点价格交易的订单，以及以零售价格改善订单形式输入交易所零售流动性计划的订单，可以以小于0.05美元的增量进行排序和接受。

Rule 67_4 收到的自动执行卖出（买入）订单将先与可显示的买入（卖出）订单进行交易，并优先传递至受保护的买入（卖出），然后再与未显示且价格相同的市价订单进行交易。交易或传递后，或两者兼而有之后，收到订单的剩余余额将按时间优先顺序抵扣所有未执行的市价订单，之后再按不可显示的平价利息进行交易。

Rule 70 根据本规则获准运营展位的会员组织不得在其获准运营的展位内为其自身账户、关联方账户或由其或其关联方在交易所行使投资酌情权的账户进行任何交易。但该会员组织可以对在场外交易公告栏或场外交易市场（OTC Bulletin Board）或场外交易市场（OTC Markets）报价的运营公司的普通股、优先股和债务证券（“OTC证券”）进行此类交易，前提是此类证券与在交易所或纽约证券交易所MKT有限责任公司（“NYSE MKT”）上市或交易的证券无关。就规则 70.40 而言，如果满足以下条件，则场外交易证券与在交易所或纽约证券交易所MKT上市或交易的证券相关：
(a) 场外交易证券由在交易所或纽约证券交易所MKT上市或交易的证券的发行人或该证券的关联方发行；
(b) 该场外交易证券受与在本交易所或纽约证券交易所MKT上市或交易的证券发行人或该发行人关联方相关的公司行为的约束；
(c) 该场外交易证券由窄基证券指数（定义见《1934年证券交易法》第3(a)(55)条）的成分股发行人发行，且该指数与在本交易所或纽约证券交易所MKT上市或交易的证券挂钩，或为纽约证券交易所MKT期权的标的；或
(d) 该场外交易证券由外国发行人发行，或为该证券的存托凭证（或同等凭证），且该外国发行人发行的证券或该证券的存托凭证（或同等凭证）在本交易所或纽约证券交易所MKT上市或交易，或为纽约证券交易所MKT期权的标的。

Rule 71 根据交易所规则发出并接受的所有买入价以及所有卖出价均具有约束力。任何以等于或高于现行卖出价的价格发出的买入价，将导致以卖出价成交，成交金额等于买入价或卖出价中较低者。当卖出价等于或低于买入价时，也适用同样的原则。

Rule 72 当会员持有买入和卖出相同数量证券的等值订单，且两个订单均为“大宗”订单，且并非由该会员或会员组织、关联人士的账户或该会员、会员组织或其关联人士行使投资酌情权的账户持有时，该会员可以以交易所最佳买入价或卖出价或其范围内的价格“交叉”这些订单。会员的买入价或卖出价应享有该交叉价的优先权，无论交易所先前显示的该价格是否存在买入价或卖出价。会员应遵循规则76的交叉程序，其他会员可与交叉交易的买入方或卖出方进行交易，但前提是该买入方或卖出方的全部或部分价格优于交叉价格。为交叉交易一方提供更优价格的会员，必须先以该价格与交易所所有其他已显示的市场参与者进行交易，然后才能进行交叉交易的任何部分。在以更优价格完成交易后，进行代理交叉交易的会员应遵循规则76的交叉程序，完成交叉交易的剩余部分。任何会员不得以交叉价格全部或部分拆分拟议的交叉交易。任何指定做市商（DMM）不得进行自营交易，以向根据本段进行的交叉交易的一方或另一方提供价格改善。根据本段以交叉价格进行的交易应打印为“已停止股票”。

Rule 72_1 在本规则中，“可显示”一词指可作为交易所 BBO 或作为其一部分发布的权益部分，包括挂钩权益。如果可显示的零散订单与在该价格点可执行的其他权益合并后，其总和等于或大于一整手，则该零散订单将作为交易所 BBO 的一部分发布。“显示权益”一词包括作为交易所 BBO 或作为其一部分发布的订单部分，其中可能包含一个或多个零散订单。

Rule 72_2 当买入或卖出（包括挂钩权益）被确定为在特定价格下发出的唯一可显示的买入或卖出，并且当该价格成为或成为交易所 BBO 时，该买入或卖出是唯一可显示的权益（“设定权益”）时，该设定权益应享有按本规则所述以该价格执行的优先分配权，但须遵守以下规定。

Rule 72_3 零股订单（包括可显示的汇总零股订单）不具备设定利息的资格。

Rule 72_4 如果在整数手或更大的可显示利息成为交易所对等价买卖（BBO）时，存在其他整数手或更大的可显示利息，包括等于或大于整数手的汇总零股订单，且价格与交易所对等价买卖（BBO）相同，则任何利息均不被视为设定利息，因此不确立优先权。

Rule 72_5 如果在整数手或更大手数的可显示权益成为交易所BBO时，存在其他可显示权益，且其总和小于整数手，且价格为交易所BBO的价格，则整数手或更大手数的可显示权益将被视为该价格点唯一可显示的买入或卖出权益，因此将确立为设定权益，并有权按照本规则规定优先分配以该价格执行的权益。

Rule 72_6 如果执行将设定权益减少至零股，则加入该剩余零股数订单的整数手或PRL订单不符合设定权益的资格。如果由于取消，权益在交易所BBO成为或成为整数手或更大手数的唯一可显示权益，则该权益将成为设定权益。

Rule 72_7 只有在交易所BBO中已公布或已公布的设定权益部分才有权优先分配执行。被指定为储备权益，因此未在交易所BBO显示（或成为交易所BBO后无法显示）的设定权益部分，无论该储备权益的价格或被交易所系统接受的时间如何，均不符合优先执行分配的资格。但是，如果在执行部分或全部设定权益后，该设定权益从任何储备权益中补充，且该设定权益仍然是交易所BBO中唯一的权益，则补充的该设定权益数量应享有优先执行权。

Rule 72_8 如果非挂钩权益成为交易所BBO，即使挂钩权益与该非挂钩权益挂钩，也应将其视为设定权益；即使随后以该价格加入一个挂钩权益，该设定权益仍应保留其优先执行权。

Rule 74 会员声称其在交易大厅内曾有先前或更佳的买入或卖出，但该买入或卖出未以必要的宣传和频率进行，以致在交易发生时未广为人知，则该会员的主张不成立。

Rule 75 买卖双方就买卖产生的争议，如未达成协议解决，应由场内官员解决。在就交易金额争议作出裁决时，场内官员应主要考虑任何非交易当事方会员的陈述，并考虑争议交易各方持有的订单规模以及其认为相关的其他事实。如果争议双方均同意，且争议涉及1万美元或以上的金额差额或交易存在争议，则可将该事项提交由三名场内主管、高级场内官员或执行场内官员或其任何组合组成的专家组裁决，其裁决对双方均具有约束力。除了由三位场内主管、高级场内官员或执行场内官员或其任何组合组成的仲裁小组外，会员还可以通过交易所规则规定的长期仲裁程序解决争议。

Rule 76 当会员同时持有同一证券的买入和卖出订单时，其在进行交易前，应以高于买入价且该证券允许的最小波动幅度的价格出价。所有此类买入和卖出均应在会员进行拟议的“交叉”交易前向交易人群清晰宣布。

Rule 77 任何会员不得在场内公开进行以下活动：
(1) 以高于或低于市场价格“止损”买卖证券；
(2) 以“收盘价”买卖证券；
(3) 买卖股息；
(4) 押注市场走势；或
(5) 买卖接收或交付证券的权利。

Rule 78 以相同价格或预估价格同时进行卖出要约，或以预估价格同时进行买回要约，均属预先安排的交易，予以禁止。本规则适用于以交易单位进行的交易以及金额较小或较大的交易。

Rule 80 (a) 如果发生一级、二级或三级市场下跌，交易所应暂停所有股票交易，并且在本规则规定的时间段内不得重新开市。
(i) 就本规则而言，市场下跌是指交易日上午 9:30 至下午 4:00 之间标普 500® 指数价格相对于前一交易日收盘价的下跌幅度。适用于该交易日的一级、二级和三级市场下跌将于上午 9:30 之前公布。
(ii) “一级市场下跌”是指市场下跌 7%。
(iii) “二级市场下跌”是指市场下跌 13%。
(iv) “三级市场下跌”是指市场下跌 20%。
(b) 交易暂停。
(i) 如果在上午 9:30 之后至下午 3:25（含）或中午 12:25（如果是提前收市的情况）期间发生一级市场下跌或二级市场下跌，交易所应在一级市场下跌或二级市场下跌发生后暂停所有股票交易 15 分钟。交易所每个交易日仅会根据一级市场下跌或二级市场下跌暂停交易一次。如果在下午 3:25 之后或中午 12:25（如果是提前收市的情况）发生一级市场下跌或二级市场下跌，交易所将不会暂停交易。
(ii) 如果在交易日的任何时间发生三级市场下跌，交易所应暂停所有股票交易，直至下一个交易日主要上市市场开市。
(c) 重新开市交易
(i) 一级或二级停牌后重新开市交易应遵循规则123D规定的程序。
(ii) 如果主要上市市场暂停所有股票交易，本交易所将暂停所有股票交易，直至主要上市市场恢复交易或收到主要上市市场可恢复交易的通知。如果主要上市市场在15分钟停牌期结束后15分钟内未重新开市，本交易所可恢复该证券的交易。
